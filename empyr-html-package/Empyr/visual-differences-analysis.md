# Visual Differences: Home 2 vs Home 3 Layouts

## Header Differences
**Identical elements:**
- Both use same "empyr" logo
- Same navigation menu (HOME, PAGES, PORTFOLIO, BLOG, CONTACT)
- Same "GET IN TOUCH" button styling

## Hero Section Layout Differences

### **Home 2 (First Image)**
- **Text arrangement:** Vertical stacking
  - "DESIGN THAT" (top line)
  - "ELEVATES" + triangle icon + "YOUR" (middle line) 
  - "DIGITAL PRESENCE." (bottom line)
- **Typography:** Larger, more spaced out text
- **Visual emphasis:** More dramatic vertical hierarchy

### **Home 3 (Second Image)**
- **Text arrangement:** Horizontal flow
  - "DESIGN THAT ELEVATES" + triangle icon (top line)
  - "YOUR DIGITAL PRESENCE." (bottom line)
- **Typography:** More compact, horizontally oriented
- **Visual emphasis:** Wider, more condensed layout

## Design Element Differences

### **Orange Circle/Oval Accent**
- **Home 2:** Positioned around "DESIGN" (left side emphasis)
- **Home 3:** Positioned around "DESIGN" (similar but slightly different angle)

### **Triangle Icon Placement**
- **Home 2:** Positioned between "ELEVATES" and "YOUR" 
- **Home 3:** Positioned at the end of "ELEVATES" line

### **Overall Layout Philosophy**
- **Home 2:** Emphasizes **vertical drama** and **bold stacking**
- **Home 3:** Emphasizes **horizontal flow** and **compact readability**

## Content Below Hero
- **Home 3:** Shows additional content visible (gray box, circular badge)
- **Home 2:** Less content visible in viewport

## Which Layout Works Better?
- **Home 2:** Better for **dramatic impact**, **mobile-friendly** (vertical stacking)
- **Home 3:** Better for **desktop viewing**, **content density**, **professional look**
